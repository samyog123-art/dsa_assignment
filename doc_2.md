# Program 2: Infix to Postfix Conversion

## Data Structure Used
Stack is used to temporarily store operators while converting infix expression to postfix.

## Functions Description
- push(): Push operator into stack.
- pop(): Remove operator from stack.
- priority(): Returns precedence of operators.

## main() Function Overview
The main function takes an infix expression as input and converts it into postfix expression.

## Sample Output

Input:
a+b*c

Output:
Postfix expression: abc*+
