# Program 3: Reverse Traversal of Singly Linked List

## Data Structure Used
Singly linked list is used where each node contains data and next pointer.

## Functions Description
- reversePrint(): Prints linked list in reverse order using recursion.

## main() Function Overview
The main function creates a linked list and displays the elements in reverse order.

## Sample Output

Input:
10 20 30

Output:
Reverse traversal: 30 20 10
