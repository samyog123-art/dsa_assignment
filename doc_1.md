# Program 1: Balanced Parentheses Using Stack

## Data Structure Used
Stack is used to store opening brackets. It follows LIFO (Last In First Out) principle.

## Functions Description
- push(): Inserts opening brackets into the stack.
- pop(): Removes the top element from the stack.
- isBalanced(): Checks whether parentheses are balanced or not.

## main() Function Overview
The main function takes an expression as input and checks whether the parentheses are balanced using stack operations.

## Sample Output

Input:
a + (b − c) ∗ (d

Output:
Expression has unbalanced parentheses

Input:
a + (b − c)

Output:
Expression has balanced parentheses
