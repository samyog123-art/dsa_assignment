
Program 8: Random Numbers Sorting (All Algorithms)

Objective:
- Generate N random integers (1-1000)
- Sort using chosen algorithm: Bubble, Selection, Insertion, Merge
- Display numbers before and after sorting
- Count comparisons and swaps

Algorithm:
1. Ask user for N
2. Generate random numbers and store in array
3. Print original numbers
4. Ask user to choose sorting algorithm
5. Apply chosen algorithm:
   - Bubble Sort: Compare & swap adjacent numbers
   - Selection Sort: Find minimum and swap
   - Insertion Sort: Place element at correct position
   - Merge Sort: Divide and merge arrays
6. Print sorted numbers
7. Display total comparisons and swaps

Variables:
- arr[]: array of numbers
- n: number of integers
- comp: total comparisons
- swap: total swaps
